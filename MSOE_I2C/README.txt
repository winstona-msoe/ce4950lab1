All library files converted to .h

Works with I2C backpack and LCD in EE2930 kit.

Copy the files to a folder name MSOE_I2C in
D:\ARDUINOworkspace

Test program "i2cportprogram.c" included.

Connect the I2C backpack CLK pin to Analog 5 (i2c SCL pin) and the I2C backpack DAT to Analog 4 (i2c SDA pin). Connect the 5V and ground pins to respective power pins.
